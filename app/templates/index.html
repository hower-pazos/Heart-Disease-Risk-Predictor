Heart Disease Prediction Model

This project aims to develop a machine learning model capable of predicting the likelihood of heart disease based on various health indicators. The model is built using Python and popular data science libraries, including Pandas, NumPy, and Scikit-learn.

Dataset:

The project uses the "heart.csv" dataset obtained from Kaggle which contains 1025 instances with 14 attributes. The attributes include demographic information (age, sex), medical measurements (resting blood pressure, cholesterol levels, etc.), and a binary target variable indicating the presence or absence of heart disease.

Model Training and Evaluation

Several machine learning models were trained and evaluated using cross-validation and test set performance: Decision Tree Classifier, Naive Bayes Classifier, Support Vector Machine (SVM), Gradient Boosting Classifier, Random Forest Classifier, and a Stacking Classifier. The models were trained on the scaled training data, and their performance was evaluated using accuracy, precision, recall, and F1-score.
